1. Servlet Lifecycle Methods

init() - Called once when servlet loads

service() - Called for every request

destroy() - Called when servlet unloads

See Lifecycle Demo

2. doGet() vs doPost()

doGet() - Retrieve/view data (idempotent)

doPost() - Submit/create data (non-idempotent)

Try creating an account (POST) and viewing it (GET)!

3. RequestDispatcher vs SendRedirect

RequestDispatcher - Server-side forward, URL doesn't change

SendRedirect - Client-side redirect, URL changes

Try the transfer operation to see RequestDispatcher in action!
